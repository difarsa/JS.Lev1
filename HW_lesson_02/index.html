var a = 1, b = 1, c, d;
1) c = ++a; alert(c); // 2
2) d = b++; alert(d); // 1
3) c = (2+ ++a); alert(c); // 5
4) d = (2+ b++); alert(d); // 4
5) alert(a); // 3
6) alert(b); // 3
Почему код даёт именно такие результаты?

var a = 1, b = 1, c, d;
1) ++a означает, что мы к переменной а прибавляем 1
2) ++ после переменной означает, что мы прибавляем единицу и выводим предыдущее значение переменной
3) в примере №1 ответ 2, потом мы прибавляем еще единицу и а становится равно 3 -> ответ 5
4) в примере №2 ответ 2, потом прибавляем еще единицу, а=3, но выводим предыдущее значение (2) -> ответ 4
5) прибавили единицу в примере №1 и №3
